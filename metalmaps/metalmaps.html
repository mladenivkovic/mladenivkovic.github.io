Metal Maps Gallery

Ever wanted to make your python plots more metal? Fear not, now you can! metalmaps delivers heavy metal and classic rock album art inspired matplotlib colormaps. For installation instructions and other details, please visit the github repository. This page provides a gallery of all available colormaps below.

But first, credit where credit is due:

  • All of this was inspired by (and shamelessly copied from) Josh Borrow's swiftascmaps
  • The "Kelvin-Helmholtz" data used in the plots below were generated using mesh-hydro.
  • The "EAGLE" data used in the plots below were obtained from the swiftsim repository.
  • The "NGC" data used in the plots below were originally obtained from flickr, credits to Judy Schmidt. I modified the image later to normalize the pixel values to be able to demonstrate the colormaps as below.
  • The album covers are linked over from musicbrainz.org.
